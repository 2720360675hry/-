## **图书管理系统**

一个基于Spring+Spring Boot（Maven方式构建）

## 项目简介

本图书管理系统基于spring，spring boot+Vue.js框架编写，前台页面使用当前主流的Element plus框架进行样式设计，数据库为MySQL。高速缓存使用Redis（使用的时候打开sever，无需安装）。

## 开发环境

前端：VS code、node.js

Java开发包：JDK 8

开发工具：IntelliJ IDEA 2022

数据库：MySQL 5.7.17

高速缓存：Redis

## 系统功能

该系统实现读者和管理员注册登录，图书的增删改查，读者的增删改查，借还图书，密码修改，上传头像，信息修改，限制访问，记录统计，密码加密等功能。

## 使用说明

1、在IDEA打开test后端代码，将数据库数据添加

2、运行Redis

3、在VS code打开AAAlibrary文件夹

4、在终端里输入npm run dev，复制Network链接，在浏览器上打开

5、根据数据库user信息登录或自己创建登入进去
